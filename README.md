## Python Inspired Machines

Pythonish to C++ Translator

### Requirements
1. Linux
2. Python 2
3. Toy Parser Generator

### Execution Instructions
``` sh
./mypi <path/to/source> <destination/folder>
```

### Example
``` sh
./mypi input/test.in demo
cd demo
make
./target
```
